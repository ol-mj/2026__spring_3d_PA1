
HW1 refined solution package

Structure
- src/: completed Python files refined for cleaner assignment alignment
- data/: sample input files copied from the conversation
- result/: output files generated by the example scripts

How to run
1) cd hw1_solution_refined/src
2) python3 T1-1_matchPicsORB.py
3) python3 T1-2_ComputeH.py
4) python3 T1-3_ComputeH_norm.py
5) python3 T1-4_RANSAC.py
6) python3 T2_planarTracker.py
7) python3 T3_panorama.py

Notes
- T1-3 and T3 default to example.jpg and example4.jpg.
- T2 uses default initial corners for the uploaded planar tracking video.
- Replace files in ../data or pass custom paths where supported.
- T1-2 now uses a deterministic, spatially spread subset of matches for plain DLT.
- T2 reports drift as mean corner reprojection error in pixels.
- T3 reuses the manual warping implementation from core_geometry.py.
