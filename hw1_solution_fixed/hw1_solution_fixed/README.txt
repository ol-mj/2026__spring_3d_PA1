
HW1 solution package

Structure
- src/: completed Python files
- data/: sample input files copied from the conversation
- result/: output files generated by the example scripts

How to run
1) cd hw1_solution_fixed/src
2) python T1-1_matchPicsORB.py
3) python T1-2_ComputeH.py
4) python T1-3_ComputeH_norm.py
5) python T1-4_RANSAC.py
6) python T2_planarTracker.py
7) python T3_panorama.py

Notes
- T1-3 and T3 default to example.jpg and example4.jpg.
- T2 uses default initial corners for the uploaded planar tracking video.
- Replace files in ../data or pass custom paths where supported.
